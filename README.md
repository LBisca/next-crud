# next-crud
SSR CRUD with material-ui

### Instruções

1 - Clone o repositório 

```
<div>
  <p>
    git clone https://github.com/LBisca/next-crud.git
    cd next-crud
  </p>
</div>
```

2 - Instale as dependências  

```
npm install
or
yarn install
```

3 - Rode o servidor

```
npm run dev
or
yarn dev
```

4 - Acesse a página em http://localhost:3000
